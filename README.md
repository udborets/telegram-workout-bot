# Start locally

## Install

### Via Anaconda

1. Create and activate Anaconda environment

``` 
conda create -n telegram-workout-bot python=3.12 -y
conda activate telegram-workout-bot
```

2. Install Python packages

```
pip install -r requrements.txt
```
